Channels

GeoFarmer uses channels to provide a project-like structure!

User Roles

In a Geo-Farmer channel, a user can have four distinct roles: a moderator, a facilitator, a geo-data manager, or a user!

Functions

Collect data and run surveys, ask questions and share knowledge!
